# react-shop

faskstore API를 이용한 반응형 쇼핑몰 제작

## 주요 기능

- 다크 모드 & 라이트 모드 변환
- 장바구니 담기 및 수량 변경
- 상품 검색

## 기술 스택

- React
- TypeScript
- Recoil
- Context
- Axios
- Vercel
